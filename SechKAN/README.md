Here are the guidelines for running the SechKAN code for the paper **"SechKAN: Kolmogorov–Arnold Networks with Hyperbolic Secant Functions."**

SechKAN performs three tasks:

- **Function fitting** (`run_ff.py` in the `main` folder)
- **Image classification** (MNIST, Fashion-MNIST, CIFAR-10, and CIFAR-100) (`run.py` in the `main` folder)
- **PDE surrogate modeling** (Navier–Stokes and Shallow Water) (`run_navier.py` and `run_shallow_water.py` in the `main` folder)
